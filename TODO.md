# nfo-viewer — Feature Roadmap

## Natural Feature Extensions

- **ANSI color parsing** — render ANSI escape sequences with proper colors (many scene NFOs use these)
- **Text search** (Ctrl+F) within the loaded NFO
- **Copy to clipboard** (as text or as rich/styled text)
- **Recent files** list
- **URL loading** — paste a URL to fetch a remote NFO
- **Font picker** — let users choose between monospace fonts (Consolas, IBM Plex Mono, Fira Code, etc.)
- **Print** support
- **Side-by-side compare** two NFOs
- **Gallery/batch mode** — browse a folder of NFOs
